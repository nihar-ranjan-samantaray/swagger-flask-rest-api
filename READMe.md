# Swagger Flask API
## Running Steps

#### 1. Create a Virtual Environment
`virtualenv venv`<br>

#### 2. Enter to Virtual Environment
`source venv/bin/activate`<br>

#### 3. Install the required Packages
`pip install -r requirements.txt`<br>

#### 4. Initialize Database
`flask db init`<br>

#### 5. Migrate Databasse
`flask db migrate`<br>

#### 6. Upgrade Database
`flask db upgrade`<br>

#### 7. Run Application

`flask run`<br>
